# 🚀 Mis Ejercicios de JavaScript (20-25)

¡Bienvenido a mi repositorio de prácticas! Este proyecto contiene una serie de ejercicios de lógica de programación desarrollados con **JavaScript Moderno (ES6+)**, organizados de forma modular.

---

## 🛠️ Tecnologías Utilizadas

*   **HTML5** - Estructura de la interfaz.
*   **CSS3** - Diseño y estilos visuales.
*   **JavaScript (ES6)** - Lógica de los ejercicios y manipulación del DOM.
*   **Git & GitHub** - Control de versiones y despliegue.

---

## 📋 Contenido de los Ejercicios

Este repositorio se enfoca actualmente en los ejercicios del **20 al 25**:

| Ejercicio | Descripción |
| :--- | :--- |
| **Ejercicio 20** | Procesamiento de cadenas y números básicos. |
| **Ejercicio 21** | Manipulación de matrices (operaciones con arreglos bidimensionales). |
| **Ejercicio 22** | Cálculos matemáticos entre dos variables. |
| **Ejercicio 23** | Unión y procesamiento de múltiples arreglos. |
| **Ejercicio 24** | Búsqueda y filtrado en arreglos numéricos. |
| **Ejercicio 25** | Operaciones avanzadas con colecciones de datos. |
